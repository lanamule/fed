2. Инициализация  и первичная настройка git для нашего проекта 

    git init - создаем локальный репозиторий

    .gitignore - Создаем файл игнорирования 

    git status - проверяем все ли файлы git видит

    git add . - добавляем файлы в локальный репозиторий

    git commit -m"start project"

----------------------------------------------------------------------------------------

    https://github.com

-----------------------------------------------------------------------------------------

4. Загрузка файлов через git на github (береться из настроек репозитория)

    git remote add origin https://github.com/OlegFD11/distfile.git
    git push -u origin master
    также можно просто загрузить файлы напрямую через web интерфейс github
